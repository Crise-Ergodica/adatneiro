Programação Orientada a Objetos

Herança - Conceito

A herança é um dos pilares da Programação Orientada a Objetos (POO). Em PHP, ela permite que uma classe (chamada de classe filha ou subclasse) herde as propriedades e métodos de outra classe (conhecida como classe pai ou superclasse). Pense nisso como uma árvore genealógica: a classe filha "herda" as características da classe pai, mas também pode ter suas próprias características exclusivas.

A grande importância da herança é a reutilização de código. Em vez de reescrever as mesmas propriedades e métodos em classes diferentes, você pode definir uma classe pai com o comportamento comum e fazer com que as classes filhas herdem esse comportamento. Isso torna o código mais prático, fácil de manter e mais organizado, pois evita a duplicação e garante que as mudanças em um local se propaguem para todas as classes filhas.

Exemplo

Código

<?php

class Funcionario {
    public $nome;
    public $salario;

    public function __construct($nome, $salario) {
        $this->nome = $nome;
        $this->salario = $salario;
    }

    public function apresentar() {
        return "Olá, sou o(a) {$this->nome} e meu salário é R$ {$this->salario}.";
    }
}

class Vendedor extends Funcionario {
    public $comissao;

    public function __construct($nome, $salario, $comissao) {
        parent::__construct($nome, $salario);
        $this->comissao = $comissao;
    }

    public function calcularSalarioTotal() {
        return $this->salario + $this->comissao;
    }
}

class Caixa extends Funcionario {
    public $bonus;

    public function __construct($nome, $salario, $bonus) {
        parent::__construct($nome, $salario);
        $this->bonus = $bonus;
    }

    public function calcularSalarioTotal() {
        return $this->salario + $this->bonus;
    }
}

$vendedor1 = new Vendedor("Carlos", 2500, 500);
echo $vendedor1->apresentar() . "<br>";
echo "Salário total do vendedor: R$ " . $vendedor1->calcularSalarioTotal() . "<br>";

$caixa1 = new Caixa("Ana", 1800, 200);
echo $caixa1->apresentar() . "<br>";
echo "Salário total da caixa: R$ " . $caixa1->calcularSalarioTotal() . "<br>";

?>

Glossário

`<?php`: Abre o bloco de código PHP.
`class Funcionario`: Declara a classe principal, `Funcionario`.
`public $nome;` e `public $salario;`: Declaram as propriedades públicas da classe.
`public function __construct(...)`: O método construtor. É executado automaticamente quando um novo objeto é criado.
`$this->nome = $nome;`: Atribui o valor do parâmetro `$nome` à propriedade `nome` do objeto atual.
`public function apresentar()`: Declara um método público que retorna uma string de apresentação.
`class Vendedor extends Funcionario`: Declara a classe `Vendedor` que herda de `Funcionario`.
`parent::__construct(...)`: Chama o construtor da classe pai (`Funcionario`).
`class Caixa extends Funcionario`: Declara a classe `Caixa` que herda de `Funcionario`.
`$vendedor1 = new Vendedor(...)`: Cria um novo objeto da classe `Vendedor`.
`echo $vendedor1->apresentar();`: Chama o método `apresentar` do objeto `$vendedor1` e exibe o resultado.
`<br>`: Tag HTML para quebra de linha.
`?>`: Fecha o bloco de código PHP.

Desafio

Crie uma hierarquia de classes em PHP para representar uma escola. A classe pai deve ser Pessoa, com as propriedades nome e idade. Crie duas classes filhas, Aluno e Professor, que herdam de Pessoa.

A classe Aluno deve ter uma propriedade adicional chamada matricula, e a classe Professor deve ter a propriedade disciplina.

Implemente um método na classe Aluno que imprima "O aluno X tem Y anos e a matrícula Z.", e na classe Professor que imprima "O professor X, de Y anos, leciona a disciplina Z.".
