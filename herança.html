Herança em Programação Orientada a Objetos (POO)

Criado por: Aurora Drumond

O que é Herança?

Imagine que você tem uma receita de bolo de chocolate. Para fazer um bolo de cenoura, você não precisa começar do zero! Você "herda" a base da receita (misturar os ingredientes, assar, etc.) e apenas adiciona os detalhes específicos do bolo de cenoura (a cenoura, a cobertura, etc.).

Na programação, a herança é a mesma coisa. É um conceito que nos permite criar uma classe "filha" que reaproveita as propriedades e comportamentos de uma classe "mãe". A classe filha pode ter suas próprias características, mas ela já começa com tudo o que a classe mãe tem.

Isso nos ajuda a evitar a repetição de código e a organizar nossas classes de forma mais lógica, seguindo uma hierarquia do "geral" para o "específico". Pense assim: um Cachorro é um Animal.

Exemplos

Exemplo 1: Animais

Aqui, Cachorro é uma classe filha que herda da classe mãe Animal. O cachorro já "nasce sabendo" de tudo que um animal sabe (como comer) e ainda tem suas próprias habilidades (como latir).

<?php

class Animal {
  public $nome;
  
  public function __construct($nome) {
    $this->nome = $nome;
  }
  
  public function comer() {
    echo $this->nome . " está comendo." . PHP_EOL;
  }
}

class Cachorro extends Animal {
  public function __construct($nome) {
    parent::__construct($nome);
  }
  
  public function latir() {
    echo "Au Au!" . PHP_EOL;
  }
}

$meuCachorro = new Cachorro("Rex");
$meuCachorro->comer(); // Saída: Rex está comendo.
$meuCachorro->latir(); // Saída: Au Au!

?>

Glossário de Termos

class Animal: Palavra-chave que define a classe principal, ou "classe mãe".

public $nome: Declara uma propriedade chamada nome. O public significa que ela pode ser acessada de qualquer lugar.

public function __construct($nome): É o método construtor. É chamado automaticamente quando um novo objeto é criado e serve para inicializar as propriedades, como o nome do animal.

$this->nome: Refere-se à propriedade nome do objeto que está sendo usado no momento.

class Cachorro extends Animal: Palavra-chave extends indica que a classe Cachorro está herdando as propriedades e métodos da classe Animal.

parent::__construct($nome): Chama o construtor da classe pai (Animal) para que ela possa fazer sua própria inicialização, passando o nome para ela.

Exemplo 2: Veículos

A classe Carro herda de Veiculo. Ela pega as características básicas (como marca e modelo) e adiciona um detalhe extra (o número de portas).

<?php

class Veiculo {
  public $marca;
  public $modelo;
  
  public function __construct($marca, $modelo) {
    $this->marca = $marca;
    $this->modelo = $modelo;
  }
  
  public function acelerar() {
    echo $this->marca . " " . $this->modelo . " está acelerando." . PHP_EOL;
  }
}

class Carro extends Veiculo {
  public $portas;
  
  public function __construct($marca, $modelo, $portas) {
    parent::__construct($marca, $modelo);
    $this->portas = $portas;
  }
}

$meuCarro = new Carro("Ford", "Ka", 4);
$meuCarro->acelerar(); // Saída: Ford Ka está acelerando.
echo "Número de portas: " . $meuCarro->portas . PHP_EOL; // Saída: Número de portas: 4

?>

Glossário de Termos

class Veiculo: Define a classe principal, a "classe mãe".

public $marca; public $modelo;: Propriedades da classe, acessíveis de qualquer lugar.

public function __construct($marca, $modelo): O construtor que recebe e inicializa a marca e o modelo.

class Carro extends Veiculo: Palavra-chave extends indica que a classe Carro herda de Veiculo.

public $portas: Uma propriedade exclusiva da classe Carro.

parent::__construct($marca, $modelo): Chama o construtor da classe pai para inicializar as propriedades que foram herdadas.
